Game Help

UI Interactions

  • Click a map node to move.
  • Click a character name to talk or attack from the popup.
  • Talk closes the popup and writes dialogue to message history.
  • Train martial arts and use items from their panels.

Combat System

Action builds over time based on agility. Active skills require enough action to use.

Inner Power is consumed by active skills and regenerates slowly during battle.

Cooldowns limit how soon an active skill can be used again.

Active Skills

断雨一击 30 inner power, 60 action - 35 damage
回灯掌 50 inner power, 40 action - restores 40 health
铁衣 40 inner power, 50 action - defense +5
听雨势 40 inner power, 50 action - enables 伞骨八刺
伞骨八刺 80 inner power, 100 action - 120 damage, requires 听雨势

Tips

  • Click character action buttons in the room panel for quick interaction
  • Click exit names to move in that direction
  • Active skills only work during combat
  • Use 听雨势 before 伞骨八刺 for maximum damage
